Portfolio

Here are some of my projects. I have worked with Support Vector Machines, Logistic Regression, Linear Regression, Nueral Networks, and many more methods.

Flower Power

Very first project I did using any kind of Neural Network. I explored various Convolutional Neural Networks to find the best network to classify on 5 different categories of flowers: Sunflowers, Roses, Daisies, Tulips, and Dandelions. Check out the code and analysis!

CNN Exploration

Here I used the pretrained VGG-16 CNN on the ImageNet dataset to visualize feature maps, saliency maps, and classes. Also, we explore how to fool our network.

AlexNet Reconstruction

Reproduced AlexNet CNN in Tensorflow to solve the famous Cats and Dogs classification problem.

LeNet 5 Reconstruction

Implemented LeNet 5 Network in TensorFlow to classify on the MNIST dataset and CIFAR 10 dataset.

Style Transfer

Here I implemented the style transfer technique. Also, generated artistic images using this method. Check it out!

Music Classification

Here we convert music files to a spectrogram to then classify using various Machine Learning methods. We try to classify artist and genre based off spectrogram.

GAN's

Here I implemented Vanilla GAN, Least-Square GAN, and Deep Convolutional GAN.

Topic Classification with SVM

Here we use SVM to perform topic classification on new blog posts based on text

Food Classification

In this project, we used various transfer learning models to create a model that accurately predicts foods from images. Then, we created an app that allows us to take any image of food and accurately predict from one of our 101 different classes.

MNIST-New Perspective

Predicting Stock Prices
